mevlog-rs TUI interface

mevlog provides a TUI interface for querying EVM transactions directly from your terminal. The TUI provides the same powerful filtering capabilities as the CLI version, with vim-style keyboard navigation.

Getting started

$ cargo install cryo_cli
$ cargo install mevlog --features=tui
$ mevlog tui

Features

  • Multi-chain - 2000+ EVM chains via ChainList RPCs, with automatic fastest-RPC selection and failover on throttling or errors.
  • Flexible filters - search by events, method calls, ENS names, validator bribes, ERC20 transfers and more.
  • Txs insights - view gas price, cost in USD, and event details.
  • EVM tracing - Revm powered EVM tracing insights that works on RPC endpoints without `debug_traceTransaction` support.
  • Vim keybindings - navigate with familiar keyboard shortcuts
